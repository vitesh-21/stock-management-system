# Stock Management System
This is web-based inventory and revenue tracking system.

The Stock Management System is designed to help small businesses track their inventory levels, manage staff logins, and monitor total revenue. It provides an easy-to-use interface for adding new stock items and viewing stock alerts.

## ✨ Key Features of the system
* **User Authentication:** Secure login for admin and staff members.
* **Inventory Tracking:** Add, edit, and delete stock items.
* **Revenue Reports:** View total sales and revenue calculations.
* **Low Stock Alerts:** Automatically identifies items that need restock.
* **User Management:** Functionality to add and manage system users.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Modern UI / Glassmorphism)
* **Backend:** PHP
* **Database:** MySQL
* **Server Environment:** XAMPP / InfinityFree

## 🚀 How to Run Locally
1.  **Clone the repository:** `git clone https://github.com/vitesh-21/stock-management-system.git`
2.  **Move to htdocs:** Place the folder into your `C:\xampp\htdocs\` directory.
3.  **Database Setup:**
    * Open `phpMyAdmin`.
    * Create a new database named `stock_db`.
    * Import the provided `.sql` file (if available).
4.  **Configure Connection:** Ensure `db.php` has the correct database credentials.
5.  **Access the app:** Open `http://localhost/stock-management-system` in your browser.
